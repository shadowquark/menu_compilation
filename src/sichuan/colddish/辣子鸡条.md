# 辣子鸡条

成菜色泽红亮，肉质酥嫩，味辣咸酸。

## 用料

鸡腿肉500克，泡辣椒75克，酱油20克，精
盐10克，料酒25克，味精1克，姜未15克，蒜末15克，葱花15克，肉汤500克，清油、醋各适量。

## 制法

1. 鸡腿去尽骨头，切成大一字条，装入盆中 用料酒、精盐拌匀，腌1小时。
2. 锅内清油烧至六成热，将鸡条放人，炸呈金黄色捞起。
3. 倒去锅内余油下泡辣椒炒出红色，加姜末、蒜末、葱花，炒出香味后加汤，然后放入鸡条，加酱油、料酒，小火烧透入味后，加味精、醋，将汁收干亮油，装托盘晾凉，装盘，浇上 烧鸡条的油汁即成。7．陈皮鸡丁】 Dicedchickenwithorangepeel 成菜色泽红亮，鸡丁酥烂，麻辣味厚，橘香浓郁。冷食热食均可，为川菜中很有特色的风味菜肴。 用料去骨仔鸡肉500克，干红辣椒15克，花椒5 克，陈皮5克，料酒15克，精盐5克，味精2克，酱油15克，醋5克，白糖5克，香油15克，姜、蒜各10克，葱25克，鸡汤400克，花生油100克。 制法（将鸡肉切成2厘米见方的丁。陈皮洗净切 块。千红辣椒去蒂、去籽，切成1.5厘米长的节。葱切节。姜切片。
4. 鸡丁加盐、料酒、姜、葱、油拌，码 味1小时。·34·
5. 将锅内花生油烧至六成热，下鸡丁炸呈 金黄色捞出。
6. 倒去锅内余油，下花椒炸出味后撇出不要，下陈皮、干辣椒，炸至棕红色，加姜，葱、蒜稍炒，然后下鸡丁，放料酒、油、白糖、味精、鸡汤（汤要淹过鸡丁），用小火煨至汁干亮油，再加醋和香油，捡去姜、葱、蒜，起锅晾凉，装盘郎成。
